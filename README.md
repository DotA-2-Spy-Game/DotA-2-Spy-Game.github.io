# DotA 2 Spy Game

Игра "Шпион" в стиле DotA 2. Игра для компании, где один из игроков - шпион, и остальным нужно его найти.

## Описание

Это веб-версия популярной настольной игры "Шпион". В DotA 2 Spy Game:
- Все игроки получают одного и того же персонажа DotA 2
- Один из игроков случайным образом становится "шпионом" и не знает, кем является
- Остальные игроки знают персонажа, но должны найти шпиона
- Игра развивает навыки наблюдения, логическое мышление и умение скрывать информацию

## Особенности реализации

- Поддержка всех 125+ персонажей DotA 2
- Сложная система fallback-ссылок для изображений персонажей
- Адаптивный дизайн для разных устройств
- Сохранение предыдущего количества игроков в localStorage
- Плавные анимации и DotA 2 тема оформления
- Обратная связь для баг-репортов и предложений

## Установка и запуск

### GitHub Pages (онлайн)
Проект готов к публикации на GitHub Pages. Просто закоммитьте и залейте в репозиторий с включёнными GitHub Pages.

### Локально
1. Клонируйте репозиторий
2. Откройте `index.html` в любом браузере

### Через Docker
```bash
./run_docker.sh
```
Или вручную:
```bash
docker-compose up -d
```
После этого сайт будет доступен по адресу http://localhost:8080
