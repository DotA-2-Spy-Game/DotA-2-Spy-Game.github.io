# DotA 2: Игра в шпиона

Интерактивная веб-игра "Шпион" с персонажами из DotA 2 для 3-12 игроков.

## Описание

Один случайный игрок становится шпионом (ему показывается "ШПИОН"), а остальные получают одного и того же случайного персонажа DotA 2. Задача обычных игроков - найти шпиона, задача шпиона - не выдать себя и угадать персонажа.

## Особенности

- Анимация переворота карточек
- Темная тема в стиле DotA 2
- Запоминание последнего количества игроков
- Адаптивный дизайн
- Поддержка всех 125 персонажей DotA 2
- Надежная загрузка изображений с резервными источниками
- Современный интерфейс с центрированием элементов

## Как играть

1. Выберите количество игроков (3-12)
2. Игроки по очереди подходят к устройству и открывают свою карточку
3. Шпион видит надпись "ШПИОН", остальные - одного и того же персонажа DotA 2
4. После того, как все посмотрят свои карточки, начинается игра

## Установка и запуск

### Вариант 1: Просто откройте в браузере
Откройте файл `dota-spy-game/index.html` в вашем веб-браузере.

### Вариант 2: Использование Docker (рекомендуется)
```bash
# Запуск с использованием docker-compose
docker-compose up -d

# Приложение будет доступно по адресу: http://localhost:8080
```

Или используйте скрипт запуска:
```bash
./run_docker.sh
```

### Вариант 3: Локальный веб-сервер
Разместите папку `dota-spy-game` в папке сервера (например, в `/var/www/html` для Apache) или используйте локальный сервер разработки.

## Технические детали

- **Frontend**: Чистый HTML/CSS/JavaScript (без фреймворков)
- **Стили**: Flexbox и CSS Grid для адаптивного дизайна
- **Изображения**: Загружаются из CDN Steam/Dota 2 с резервными источниками
- **Состояние игры**: Управление через JavaScript
- **Тема**: Темная, в стиле DotA 2 с акцентными цветами

## Поддержка персонажей

Игра поддерживает всех 125 персонажей DotA 2 с корректными именами и надежной загрузкой изображений. Включает специальные обработки для персонажей с отличающимися внутренними именами, такими как:
- Shadow Fiend → nevermore
- Wraith King → skeleton_king/wraith_king
- Nature's Prophet → furion
- Windranger → windrunner
- Underlord → abyssal_underlord
- и другие

## Разработка

### Тестирование изображений
Файл `dota-spy-game/test-images.html` может быть использован для проверки загрузки изображений всех персонажей и отладки проблем с изображениями.

### Стили
CSS использует современные методы центрирования и адаптивного дизайна, включая:
- Flexbox для центрирования элементов
- CSS Grid для компоновки
- Responsive design для различных размеров экрана
- Анимации при перевороте карточек

### JavaScript
- Управление состоянием игры
- Логика выбора шпиона и персонажа
- Обработка событий и анимаций
- Система резервных источников изображений

## Безопасность и производительность
