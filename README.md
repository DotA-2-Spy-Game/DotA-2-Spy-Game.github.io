# DotA 2: Игра в шпиона

Интерактивная веб-игра "Шпион" с персонажами из DotA 2 для 3-12 игроков.

## Описание

Один случайный игрок становится шпионом (ему показывается "ШПИОН"), а остальные получают одного и того же случайного персонажа DotA 2. Задача обычных игроков - найти шпиона, задача шпиона - не выдать себя и угадать персонажа.

## Как играть

1. Выберите количество игроков (3-12)
2. Игроки по очереди подходят к устройству и открывают свою карточку
3. Шпион видит надпись "ШПИОН", остальные - одного и того же персонажа DotA 2
4. После того, как все посмотрят свои карточки, начинается игра

## Запуск

Откройте файл `index.html` в браузере или запустите локальный сервер.

## Особенности

- Анимация переворота карточек
- Темная тема в стиле DotA 2
- Запоминание последнего количества игроков
- Адаптивный дизайн
